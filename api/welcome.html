Endpoints served by this API

General notes

Anytime 'Paginated' is mentionned as true, This means, that the request needs the X-Range Header to be specified with the start and end included indexes specifying the range that API user needs to fetch from given collection.

The X-Lang header can be filled with language codes, that aren't ISO codes, but simple 'conventional' 2 letter codes. Mostly used as of 2015 are 'fr', 'en' and 'es', but you're welcome on http://plugins.glpi-project.com/#/submit to read about the XML Glpi Description File specification, which allows you to declare tags to promote your plugin in your native language, and, to provide description of the plugin in as many language as you want, anywhere you're from. This header change some special properties of resources fetched, and will use translation if possible on specific fields. An example is in /search , where short_description of plugin will be given in the specified language, if available, in the plugin JSON object returned. A fallback to English ('en' code) is default behavior if no X-Lang is specified. On a lot of endpoints, X-Lang has no effect. It is also useful to mention that each time a plugin is requested via /plugin/<id>, all descriptions (in all languages) and all tags (in all languages) are sent, this means that the api user (being a software or a cURL user) needs to filter the data he needs. the X-Lang header then acts as a shortcut to have some specially translated properties added to the scope of the json object being returned.

Contributors

  • Contributor list

    URL: /author Method: GET Description: List of GLPI known contributors Paginated: true (answers 206 Partial Content)

    Example usage (HTTP Session) 

                   
                  GET /api/author HTTP/1.1
                  X-Lang: en
                  X-Range: 0-14

    Example usage (cURL call) 

                   
                  curl -X GET -H 'X-Lang: en' -H 'X-Range: 0-14' http://plugins.glpi-project.com/api/author

  • Contributor card

    URL: /author/<id> Method: GET Description:Description of a GLPI contributor

    Parameters

    • Name: id Description: ID of the GLPI contributor to look at
    Paginated: false

    Example usage (HTTP Session) 

                   
                  GET /api/author/3 HTTP/1.1
                  X-Lang: en

    Example usage (cURL call) 

                   
                  curl -X GET -H 'X-Lang: en' http://plugins.glpi-project.com/api/author/3

  • Contributor plugin list

    Paginated: true

    Example usage (HTTP Session)

Plugins

  • Search

    URL: /search Method: POST
    • Name: query_string Description: Search string to query
    Description: The collection of all GLPI known plugins matching the query_string Paginated: true (answers 206 Partial Content)

    Example usage (HTTP Session) 

                   
                  POST /api/search HTTP/1.1
                  Content-Type: application/json
                  X-Lang: en
                  X-Range: 0-14

                  {"query_string": "additional"}

    Example usage (cURL call) 

                   
                  curl -X POST -H 'X-Lang: en' -H 'X-Range: 0-14' -H 'Content-Type: application/json' -d '{"query_string": "additional"}' http://plugins.glpi-project.com/api/search
