Endpoints served by this API

Contributors

  • Contributor list

    URL: /author Method: GET Description: List of GLPI known contributors Paginated: true (answers 206 Partial Content)

  • Contributor card

    URL: /author/<id> Method: GET Description:Description of a GLPI contributor

    Parameters

    • Name: id Description: ID of the GLPI contributor to look at

  • Contributor plugin list

    URL: /author/<id> Method: GET Description: Collection of plugins the contributor authored on

    Parameters

    • Name: id Description: ID of the GLPI contributor to look at

Plugins

  • Search

    URL: /search Method: GET Method: POST
    • Name: query_string Description: Search string to query
    Description: The collection of all GLPI known plugins matching the query_string Paginated: true (answers 206 Partial Content)

  • Plugin list

    URL: /plugin Description: The collection of all GLPI known plugins Paginated: true (answers 206 Partial Content)

  • Trending Plugin list

    URL: /plugin/trending Description: Top 10 downloaded in this month Paginated: true (answers 206 Partial Content)

  • Popular Plugin list

    URL: /plugin/popular Description: Top 10 downloaded/voted plugins Paginated: true (answers 206 Partial Content)

  • Updated Plugin list

    URL: /plugin/updated Description: Top 10 recently updated plugins (by change of XML Plugin Description file) Paginated: true (answers 206 Partial Content)

  • New Plugin list

    URL: /plugin/new Description: The collection of all GLPI known plugins Paginated: true (answers 206 Partial Content)

  • Plugin card

    URL: /plugin/<id> Method: GET Description:Description of a GLPI contributor

    Parameters

    • Name: id Description: ID of the GLPI plugin to look at

Tags

  • Tag list

    URL: /tags Method: GET Description: The collection of all known plugin tags Paginated: true (answers 206 Partial Content)

  • Tag card

    URL: /tags/<key> Method: GET Description: Description infos of a tag

    Parameters

    • Name: key Description: identified name of the tag
